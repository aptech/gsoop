Detailed Description

This is the callback function that GAUSS will call to do normal program output.

This same callback is also used for error output as well, since a string is passed to both.

You will have to provide your own implementation.

Note: With all callbacks, the thisown flag must be set to 0 on instantiation. This informs the target language that this object will be owned and properly deleted by the Engine.

PHP

class Output extends IGEProgramOutput {
function invoke($message) {
echo $message;
}
}
$out = new Output();
$out->thisown = 0;
$ge->setProgramOutput($out);
$ge->executeString("rndseed 12345;");
$ge->executeString("rndu(3, 3);");

will result in the output:
