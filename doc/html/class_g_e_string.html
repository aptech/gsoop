Detailed Description

GAUSS String Symbol type.

For all intents and purposes, this is a standard string wrapper.

string GEString::getData ( )

Return a copy of the data as a string.

Python

1 ge.executeString("s = \"Hello World!\"")
2 s = ge.getString("s")
3 print "s = " + s.getData() # str(s) would be identical

PHP

$ge->executeString("s = \"Hello World!\"");
$s = $ge->getString("s");
echo "s = " . $s->getData() . PHP_EOL; // Specifying '$s' would be identical

would result in output:

s = Hello World!
