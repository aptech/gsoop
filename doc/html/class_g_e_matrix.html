Detailed Description

GAUSS Matrix symbol type.

Represents two dimensional array of double precision numbers. Matrices can also be complex, storing both real and imaginary data.

vector< double > GEMatrix::getData ( bool  imag = false) const

Retrieve a copy of underlying numeric vector.

Imaginary data can also be queried by supplying the imag argument, thus appending it to the end of the real data. Omitting or specifying the imag argument as false will return only real data. If you wish to only view the imaginary data, there is a getImagData() method for convenience.

Example:

Python

1 # Create a 4x2 matrix of integers 1 - 8.
2 ge.executeString("x = reshape(seqa(1, 1, 8), 4, 2)")
3 ge.executeString("print x")
4 x = ge.getMatrix("x")
5 print ", ".join(str(n) for n in x.getData())

PHP

// Create a 4x2 matrix of integers 1 - 8.
$ge->executeString("x = reshape(seqa(1, 1, 8), 4, 2);");
$ge->executeString("print x;");
$x = $ge->getMatrix("x");
echo implode(", ", $x->getData());

results in output:

1.0000000 2.0000000
3.0000000 4.0000000
5.0000000 6.0000000
7.0000000 8.0000000
1, 2, 3, 4, 5, 6, 7, 8

vector< double > GEMatrix::getImagData ( ) const

Retrieve a copy of underlying imaginary data storage.

Example:

Python

1 # Create two 2x2 matrices and combine to form a complex matrix.
2 ge.executeString("x_real = reshape(seqa(1, 1, 4), 2, 2)")
3 ge.executeString("x_imag = reshape(seqa(5, 1, 4), 2, 2)")
4 ge.executeString("x = complex(x_real, x_imag)")
5 x = ge.getMatrix("x")
6 print ", ".join(str(n) for n in x.getImagData())

// Create two 2x2 matrices and combine to form a complex matrix.

$ge->executeString("x = complex(x_real, x_imag);");
