Detailed Description

GAUSS String Array Symbol type.

Represents a standard string array. Data is stored internally as a vector.

Constructor & Destructor Documentation

GEStringArray::GEStringArray ( vector< string >  data)

Initialize a one-dimensional string array with a user specified vector.

This assumes that there are data.size() columns and 1 row.

Example:

Python

1 sa = GEStringArray(["Open", "High", "Low"])
2 print sa

PHP

$sa = new GEStringArray(array("Open", "High", "Low"))
echo $sa;

results in output:

Open High Low

vector< string > GEStringArray::getData ( )

Return a collection copy of strings as a vector.

Example:

Python

1 # Create a string array using GAUSS
2 ge.executeString("string sa = { one two three four, five six seven eight }")
3 
4 # Retrieve the string array from the symbol table
5 sa = ge.getStringArray("sa")
6 
7 print sa
8 print " ".join(sa.getData())

PHP

// Create a string array using GAUSS
$ge->executeString("string sa = { one two three four, five six seven eight };");
// Retrieve the string array from the symbol table
$sa = $ge->getStringArray("sa");
echo $sa . PHP_EOL;
echo implode(" ", $sa->getData()) . PHP_EOL;

resulting in the output:

ONE TWO THREE FOUR
FIVE SIX SEVEN EIGHT
ONE TWO THREE FOUR FIVE SIX SEVEN EIGHT
