\mbox{\hyperlink{class_g_a_u_s_s}{GAUSS}} Matrix symbol type.

Retrieve a copy of underlying numeric vector. 

Imaginary data can also be queried by supplying the {\itshape imag} argument, thus appending it to the end of the real data. Omitting or specifying the {\itshape imag} argument as {\ttfamily false} will return only real data. If you wish to only view the imaginary data, there is a \mbox{\hyperlink{class_g_e_matrix_a7610a3e67e0729c3a1d4ae5648d2aae5}{get\+Imag\+Data()}} method for convenience.
